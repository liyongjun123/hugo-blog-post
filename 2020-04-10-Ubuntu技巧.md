## # 快捷键

文件管理器

```bash
Ctrl + 1	// 列表形式显示
Ctrl + 2	// 图标形式显示
```

top指令

```bash
Shift + P	// 将进程按照CPU占用，从大到小排序
Shift + M	// 将进程按照内存占用，从大到小排序
```

命令行终端

```bash
F1	// 打开帮助指南
F11	// 全屏切换

Ctrl + Alt + T	// 新建终端
Ctrl + Shift + N	// 新建终端，路径停留在当前路径(当前焦点窗口必须是终端)

Ctrl + D			// 关闭终端
Ctrl + Shift + D	// 关闭终端

Ctrl + Shift + T	// 新建标签页
Ctrl + Shift + W 	// 关闭标签页

Ctrl + Shift + C	// 复制
Ctrl + Shift + V	// 粘贴

Ctrl + A	// 将光标移动到开头
Ctrl + E	// 将光标移动到末尾

Ctrl + 左键头	// 光标移动到上一个单词的词首
Ctrl + 右键头	// 光标移动到下一个单词的词尾

Ctrl + W	// 向前删除一个单词

Ctrl + U	// 删除此处到开头的所有内容
Ctrl + K	// 删除此处到末尾的所有内容

Ctrl + Y	// 粘贴由 Ctrl + U / Ctrl + K / Ctrl + w 删除的内容
Ctrl + &	// 恢复由 Ctrl + U / Ctrl + K / Ctrl + w 删除的内容


Ctrl + S	// 锁屏
Ctrl + Q	// 解除锁屏


```

浏览器

```bash
Ctrl + T	// 新建标签页
Ctrl + W	// 关闭标签页



```
