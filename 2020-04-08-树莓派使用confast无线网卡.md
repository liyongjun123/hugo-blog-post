插在linux机器上无法使用，联系卖家，要装驱动，并给了一个驱动链接：

下载后编译不成功

网上说用这个驱动可以：https://github.com/brektrou/rtl8821CU.git

就下下来编译：

提示没有kernel headers,

安装

```
sudo apt install raspberrypi-kernel-headers
```
